- **Gin**: 高性能 HTTP 框架，用于构建 RESTful API 服务
    
- **github.com/mattn/go-sqlite3**:  SQLite 驱动，用于连接和操作 SQLite 数据库
    
- **GORM**: ORM 库，简化 Go 结构体与 SQLite 表的映射及 CRUD 操作
    
- **github.com/golang-jwt/jwt**:  JWT 鉴权库，用于用户身份认证与令牌生成/解析
    
- **github.com/google/uuid**:  UUID 生成库，用于关卡、用户、成就等唯一 ID 生成
    
- **encoding/json**: 标准库 JSON 编解码，用于将关卡蓝图渲染为前端可读的 JSON
    
- **github.com/xeipuuv/gojsonschema**: JSON Schema 验证库，用于校验和生成前后端交互的 Schema
    
- **go.uber.org/zap**: 结构化日志库，用于高性能日志记录与分级输出
    
- **github.com/prometheus/client_golang**: Prometheus 客户端，用于埋点指标收集及监控
    
- **github.com/go-playground/validator/v10**: 字段验证库，用于校验请求参数和模型合法性
    
- **github.com/gorilla/websocket**: WebSocket 库，用于实时推送成就通知至前端
    
- **gonum.org/v1/gonum/stat**: 统计分析库，用于计算正确率、放弃率等行为数据指标
    
- **github.com/spf13/viper**: 配置管理库，用于加载和管理多环境配置文件
    
- **github.com/robfig/cron/v3**: 定时任务库，用于周期性生成学习报告和复习推荐
    
- **github.com/ethereum/go-ethereum**: 以太坊客户端库，用于 NFT 链上交互与铸造 (可选)
